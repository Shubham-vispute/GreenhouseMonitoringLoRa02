About This Project

This greenhouse monitoring system uses IoT technologies such as:

  • ESP32 with sensors (DHT22, Soil Moisture, MQ135, TSL2561)
  • LoRa for long-range wireless communication
  • ESP8266 as a gateway to Firebase Realtime Database
  • Flask web server for data visualization and relay control

Designed for energy-efficient, remote-controlled greenhouse environments.
